# Weather App

## Overview

A responsive weather application that fetches and displays real-time weather information using the OpenWeatherMap API.

Users can search for any city and view current weather conditions including temperature, humidity, wind speed, weather description, and weather icons.

## Features

* Search weather by city name
* Real-time weather data
* Temperature display
* Humidity display
* Wind speed display
* Feels-like temperature
* Weather condition icons
* Clean and responsive user interface

## Technologies Used

* HTML
* CSS
* JavaScript
* OpenWeatherMap API

## How to Run

1. Clone the repository
2. Open the project in VS Code
3. Add your OpenWeatherMap API key in script.js
4. Open index.html in a browser
5. Enter a city name and click Search
